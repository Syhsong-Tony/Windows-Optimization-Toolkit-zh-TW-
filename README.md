\# Windows Optimization Toolkit (zh-TW)



\## 簡介 (Introduction)

這是一個專為 \*\*Windows 桌面環境\*\* 設計的批次檔工具，能快速清理暫存檔、快取、縮圖、DNS、Print Spooler 等，並提供系統檔案檢查與 DISM 健康檢查。

This is a \*\*Windows desktop optimization toolkit\*\* written in batch script, designed to quickly clean temporary files, caches, thumbnails, DNS, Print Spooler, and also run system file check and DISM health check.



Keyword: 「Windows Optimization」「Batch Script」「System Cleanup」「Performance Tuning」

\---



\## 功能特色 (Features)

\- 🧹 清理暫存檔、Prefetch、DNS 快取

\- 🔄 重啟 Explorer、重置背景程序

\- 🖨️ Print Spooler 清理

\- 🛠️ 系統檔案檢查 (sfc /scannow)

\- 🩺 DISM 健康檢查

\- 💾 磁碟清理 (cleanmgr)

\- 🌐 瀏覽器快取清理 (Edge/Chrome/Firefox)

\- 📂 最近使用檔案紀錄清理

\- 🖼️ 縮圖快取清理

\- 🚀 主選單介面，支援倒數自動執行全部步驟



\---
